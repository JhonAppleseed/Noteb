Noteb 📝
A simple, private web-based note-taking application built for friends and those who need a clean space to store thoughts.

Features
Private Accounts: Create a username and password to access your personal vault.

Cloud Storage: Your notes are stored online and accessible from anywhere.

Lightweight: No bloated features, just you and your notes.

Tech Stack
Frontend: React

Backend: Python (Flask/FastAPI)

Database: SQLite

Getting Started
Prerequisites
Node.js and npm

Python 3.x

Installation
Clone the repo:

Bash
git clone https://github.com/yourusername/noteb.git
cd noteb
Setup Frontend:

Bash
npm install
npm start
Setup Backend:
Navigate to the root folder and run:

Bash
python run.py
